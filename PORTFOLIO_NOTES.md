## Interview Explanation

I took a basic serverless Todo app and upgraded it into a production-style DevOps project. The app uses React for the frontend, API Gateway and Lambda for the backend, and DynamoDB for storage. I kept the application functionality simple, but improved the engineering around it using Terraform modules, GitHub Actions pipelines, OIDC-based AWS authentication, CloudWatch logging and alarms, DynamoDB point-in-time recovery, and CloudFront security headers.

I also added modern DevSecOps practices like Trivy filesystem scanning, CodeQL analysis, Checkov for Terraform security checks, Dependabot, SBOM generation, and a drift detection workflow. To align it with current AI operations trends, I added an optional offline AI-ready release summary script that can later be extended to Bedrock or another approved GenAI service.
